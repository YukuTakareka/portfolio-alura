Circulo Interativo

Este projeto é uma animação interativa desenvolvida com p5.js, onde um círculo colorido segue suavemente o cursor do mouse pela tela. Além do movimento dinâmico, o círculo muda de cor aleatoriamente toda vez que o botão do mouse é pressionado, criando um efeito visual atrativo e responsivo.

A proposta do projeto é explorar conceitos de interação com o usuário, como detecção de posição do mouse, animação quadro a quadro e manipulação de cores aleatórias, oferecendo uma experiência visual simples, mas envolvente.

Recomendador de Filmes Interativo

Este projeto é um sistema de recomendação de filmes interativo, desenvolvido com a biblioteca p5.js, que oferece sugestões personalizadas com base na idade do usuário e nas preferências por fantasia e aventura. A interface é simples, visual e responsiva, permitindo uma experiência amigável diretamente no navegador.

Ping Pong Interativo

Este projeto é uma recriação clássica do jogo Ping Pong, desenvolvida com a biblioteca p5.js, que permite criar experiências visuais e interativas diretamente no navegador. O jogo simula uma partida entre o jogador e um oponente controlado pela IA, com placar em tempo real, colisões, movimentação fluida e interação com teclado.

Com uma interface simples e intuitiva, o jogo reforça conceitos fundamentais de programação orientada a eventos, física básica (colisão e movimento) e inteligência artificial simples para o comportamento do oponente.
